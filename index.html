QuerySweep

A DBeaver plugin that runs one SQL query against many connections in parallel, with color-coded results.

This page is the p2 update site. To install in DBeaver:

  1. Help → Install New Software…
  2. Paste this URL into Work with and press Enter:

    https://mlorek.github.io/query-sweep/

  3. Check QuerySweep Multi-Connection Query, click Next → Finish, accept the unsigned-content warning, and restart DBeaver.

Source, docs, and issues: github.com/mlorek/query-sweep
